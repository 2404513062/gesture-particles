# 🌟 手势控制粒子交互系统

> 用摄像头识别手势，触发华丽粒子特效 — 桌面版 + 手机网页版

## 📱 手机网页版（推荐发给好友！）

**无需安装任何东西，浏览器打开就能用！**

### 🚀 快速体验
1. 双击运行 **`mobile_server.bat`** 启动本地服务器
2. 手机和电脑连接**同一个 WiFi**
3. 手机上打开浏览器，访问屏幕上显示的地址
4. 允许摄像头权限，开始玩！

### 🔗 部署到互联网（好友远程打开）
把 `gesture_particles.html` 上传到任意静态托管：
- **GitHub Pages**（免费）：上传到仓库 → Settings → Pages → 得到链接
- **Netlify**（免费）：拖拽 HTML 文件到网页即可
- 把链接发给好友，对方浏览器打开即用

## 💻 桌面版（Python）

### 📋 使用前准备
- **Python 3.8+**：[python.org](https://www.python.org/downloads/) 下载
- **一个摄像头**（笔记本自带的就可以）
- **Windows 10/11**

### 🚀 快速开始
1. **双击 `install_and_run.bat`** → 自动安装依赖并启动
2. 或者手动：
```bash
pip install -r requirements.txt
python gesture_particles.py
```

### 🎮 按键控制
| 按键 | 功能 |
|------|------|
| `Q` / `Esc` | 退出 |
| `H` | 显示/隐藏帮助 |
| `1`~`7` | 手动切换粒子效果 |
| `0` | 切回自动识别 |
| `R` | 清除所有粒子 |
| `Space` | 截图保存 |

## ❓ 常见问题

**Q: 手机打开网页摄像头不工作？**
A: 确保已允许浏览器摄像头权限。iOS 用户请用 Safari 打开，微信内请点"在浏览器中打开"。

**Q: 微信内置浏览器打不开？**
A: 微信限制了摄像头权限，请点击右上角 → "在浏览器中打开"。

**Q: 手势识别不准？**
A: 确保手在光线充足的环境中，手掌正对摄像头，动作干脆一些。

**Q: 程序卡顿？**
A: 桌面版可调低 `CAMERA_WIDTH/HEIGHT`。手机版自动优化（粒子数上限 180）。
